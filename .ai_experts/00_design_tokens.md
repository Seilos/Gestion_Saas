# Tokens de Diseño: Estándar Visual ClaraPos

## Perfil
Este no es un experto, sino la "Fuente Única de Verdad" para el diseño. Define los valores constantes que el `02_frontend_visionary` y el `06_qa_debugger` deben respetar para mantener la consistencia.

## Colores Primarios (Bootstrap Custom)
- **Primary (Core)**: `#0d6efd` -> Personalizado a `#1a73e8` (Azul Google/Professional).
- **Dark (Sidebar)**: `#212529` -> Personalizado a `#15181b` (Profundo).
- **Accent (Info)**: `#0dcaf0` -> Usado para acciones secundarias y estados vivos.
- **Success**: `#198754` -> Para estados positivos y ventas exitosas.

## Tipografía (Google Fonts)
- **Principal**: `Outfit` (Headers y Brand).
- **Cuerpo**: `Inter` (Lectura y Datos).
- **Código/IDs**: `JetBrains Mono` o `Roboto Mono`.

## Espaciado y Formas
- **Border Radius**: Estándar `rounded-3` (8px). Botones `rounded-pill` para acciones de marketing.
- **Shadows**: `box-shadow: 0 4px 6px -1px rgba(0,0,0,0.1), 0 2px 4px -1px rgba(0,0,0,0.06);`.
- **Gaps**: Uso estricto de múltiplos de 4px (Bootstrap `g-1` a `g-5`).

## Componentes UI Estándar
- **Sidebar**: Ancho `250px`, Fixed, Fondo Dark.
- **Cards**: Fondo blanco, borde suave, sin sombras agresivas.
- **Botones**: Altura mínima `42px` para accesibilidad móvil.

"La consistencia es lo que separa una app amateur de una plataforma profesional."
